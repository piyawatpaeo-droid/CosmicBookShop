CosmicBookShop
CosmicBookShop เป็นโปรแกรมสำหรับการขายหนังสือแบบ Desktop Application
ใช้สำหรับจัดการข้อมูลหนังสือในร้าน และออกใบเสร็จสำหรับลูกค้า

คุณสมบัติของโปรแกรม
โปรแกรมรองรับการจัดการข้อมูลหนังสือและการขาย ดังนี้:

ปุ่มการทำงานหลัก
AddBook
ใช้ในการเพิ่มหนังสือใหม่เข้าสู่ตารางข้อมูล
DeleteBook
ใช้ในการลบข้อมูลหนังสือออกจากตาราง
EditBook
ใช้ในการแก้ไข/ปรับปรุงข้อมูลหนังสือในตาราง
Receipt
ใช้ในการออกใบเสร็จหลังจากทำรายการขาย

เทคโนโลยีที่ใช้
ภาษา: C#
รูปแบบโปรแกรม: Windows Forms
ฐานข้อมูล: SQLite

โครงสร้างพื้นฐานของระบบ
ระบบจัดเก็บข้อมูลหนังสือในรูปแบบตาราง
สามารถเพิ่ม แก้ไข และลบข้อมูลได้แบบ Real-time
รองรับการออกใบเสร็จสำหรับลูกค้า

วัตถุประสงค์ของโปรเจกต์
เพื่อฝึกการพัฒนาโปรแกรมด้วย C#
เพื่อเรียนรู้การเชื่อมต่อฐานข้อมูล SQLite
เพื่อสร้างระบบจัดการร้านหนังสือเบื้องต้น
